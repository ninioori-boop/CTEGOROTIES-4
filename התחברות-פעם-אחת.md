# התחברות ל-GitHub פעם אחת – אחר כך בלי כניסה

אם תגדיר **פעם אחת** שמירת פרטי התחברות, then `git push` יעבוד בלי לבקש סיסמה כל פעם.

---

## איך לעשות (בחר אחת)

### אפשרות 1: Git Credential Manager (כבר מגיע עם Git for Windows)
1. הרץ **פעם אחת**: `git push origin main`
2. כשיפתח חלון – **התחבר** עם חשבון **ninioori-boop**
3. סמן **"Remember me"** / **"Store credential"** אם מופיע
4. מהפעם הבאה – לא יבקש שוב

### אפשרות 2: Personal Access Token (אם אין חלון התחברות)
1. ב-GitHub: **Settings** → **Developer settings** → **Personal access tokens** → **Generate new token**
2. תן שם, סמן scope **repo**, צור והעתק את ה-Token
3. כשמריצים `git push` ויבקשו **Password** – הדבק את ה-**Token** (לא את סיסמת החשבון)
4. אם יבקש "Store credential?" – ענה **yes**

### אפשרות 3: GitHub Desktop
- התקן GitHub Desktop, התחבר פעם אחת – ואז כל Push מהתוכנה יעבוד בלי כניסה.

---

אחרי שזה מוגדר – כשתבקש "העלה לגיטהאב" תקבל רק את הפקודות, תריץ אותן, והדחיפה תעבוד בלי להזין פרטים.
