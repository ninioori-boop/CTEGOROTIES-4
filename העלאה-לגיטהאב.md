# העלאה לגיטהאב – המקטרג (CTEGOROTIES-4-main)

## מה המצב עכשיו
בתיקייה **אין** עדיין מאגר Git. צריך לאתחל ולחבר ל-GitHub פעם אחת.

---

## מה לעשות (פעם אחת)

### 1. לפתוח Terminal בתיקיית הפרויקט
- ב-Cursor: **Terminal → New Terminal**
- או לפתוח CMD/PowerShell ולנווט לתיקייה:
  ```
  cd "c:\Users\ninio\OneDrive\הכלכלן של הבית\CTEGOROTIES-4-main"
  ```

### 2. לאתחל Git ולהוסיף קבצים
```bash
git init
git add .
git commit -m "חיזוק זיהוי תשלומים בדוח מיפוי"
```

### 3. ליצור ריפו חדש ב-GitHub
- היכנס ל־https://github.com/new
- שם הריפו (למשל): `mekatreg` או `CTEGOROTIES-4-main`
- **אל תסמן** "Add a README" (כבר יש קבצים אצלך)
- לחץ Create repository

### 4. לחבר את התיקייה ל-GitHub ולדחוף
החלף `YOUR_USERNAME` ו-`REPO_NAME` בשם המשתמש ובשם הריפו שיצרת:

```bash
git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git
git branch -M main
git push -u origin main
```

אם GitHub попросит התחברות – התחבר (או השתמש ב־GitHub Desktop / Token).

---

## אחרי שזה עבד
מעכשיו, כשתבקש "העלה לגיטהאב" או "דחוף שינויים", אוכל להריץ:

```bash
git add .
git commit -m "תיאור השינוי"
git push
```

(בתנאי ש-Git מותקן וזמין ב-Path ב-Cursor.)

---

## אם Git לא מוכר ב-Terminal
- התקן Git: https://git-scm.com/download/win  
- אחרי ההתקנה – **סגור ופתח מחדש** את Cursor, ואז נסה שוב את הפקודות.
